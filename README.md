## Playing Cards
- This is a coding exercise that draws cards to the total number of players

### Running the App
1. Run `nvm use` to use the node version I used(8.10)
1. Install node packages `npm install` or `yarn install`
2. Run the application by issuing `npm run start (input value)`
3. eg. `npm start 3`

### Utility Commands
>`npm run lint`

I use eslint and I added some rules which can be seen in `.eslintrc`
